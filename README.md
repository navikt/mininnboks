Mininnboks
================

## Utvikling

`npm i`
`npm start`

## Test

`npm run build`

## Tips

* Marker node_modules mappen som ekskludert i IntelliJ. Høyreklikk på mappen, velg "Mark Directory As" og "Excluded".
